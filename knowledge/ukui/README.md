# UKUI 知识章节

本目录记录 UKUI 桌面功能、服务、设置后端和组件定制相关知识。

- `autostart.md`：开机自启动、设置界面显示、`.desktop`、图标解析和 `sort-app-list`。
- `keybindings.md`：全局快捷键、系统占用提示、全局搜索快捷键和窗口菜单冲突。
- `search.md`：全局搜索来源、软件商店结果、搜索引擎设置和相关 D-Bus 插件。
- `system-tray.md`：右侧托盘顺序、隐藏区、`systemTray.json` 和持久化行为。
- `system-service-manager.md`：桌面服务启动顺序、D-Bus activation、孤儿进程和面板/任务栏异常。
- `kylin-ai-subsystem.md`：桌面 AI 助手、AI 子系统卸载边界和残留清理。
